# Architecture Decision Records

This directory stores Architecture Decision Records (ADRs) for the Vibe Plan
platform.

ADR status values:

- `Proposed`: under review and expected to change.
- `Accepted`: approved as the current direction.
- `Superseded`: replaced by a later ADR.

New ADRs should use the filename pattern:

`NNNN-short-kebab-case-title.md`
